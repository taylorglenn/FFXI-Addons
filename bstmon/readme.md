# bstmon #

* A Final Fantasy XI Beastmaster Utility

#### Command Prefix: ####
//dnc

#### Commands: ####
1.  show - Shows the addon.
2.  hide - Hides the addon.
3.  verbose - Shows or hides the names of ready moves.
4.  ready <ready_move_index> - Uses the ready moe with the given index
